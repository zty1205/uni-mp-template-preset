# uni-mp-template

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Customize configuration

See [Configuration Reference](https://cli.vuejs.org/config/).

<br/>

## 模板功能

- 支持 appid 替换
- 支持环境变量
- 支持分包 copy-wxcomponents
- 支持 ts
- 通用方法封装：埋点，微信相关，通用方法
- toast loading 优先级处理
- 通用性网关
- store 基本模板
- commit 规范化
